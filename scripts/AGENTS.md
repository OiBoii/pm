# Scripts Notes

This folder contains Docker start/stop scripts for local development.

## Linux

- `start-linux.sh`
- `stop-linux.sh`

## macOS

- `start-mac.sh`
- `stop-mac.sh`

## Windows

- `start-windows.ps1` / `stop-windows.ps1`
- `start-windows.bat` / `stop-windows.bat` wrappers

All scripts manage the `pm-mvp-app` container and expose the app on port `8000`.
